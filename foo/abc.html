# Forcepoint客户端安装手册

# 生成Forcepoint Endpoint客户端

  • 拷贝Builder包到FSM服务器，并解压Builder包；

  1. 拷贝打包文件到以下文件夹进行替换 C:\Program Files (x86)\Websense\Data Security\client；

  2. 双击【WebsenseEndpointPackageBuilder】打开客户端生成程序；

  3. 在打开的“Forcepoint One Endpoint Package Builder”对话框中，选择【Forcepoint DLP Endpoint】复选框，并且在下方选择要安装的语言版本；

  4. 在“Installation Platform and Security”页面中，选中【Windows 32-bit】、【Windows 64-bit】和【Mac】复选框，并且选中【Protect installation directory from modification or deletion】复选框；

  5. 在“Installation Path”页面，选择默认即可；

  6. 在“Forcepoint One Endpoint：DLP Server Connection”页面，在上方文本框中输入用于分发Policy及Profile的FSM服务器IP地址（如172.31.0.55）；

  7. 在“Forcepoint One Endpoint：DLP Client Setting”页面中，选中默认的【Interactive】及【Full】单选框；

  8. 在“Save Installation Package”页面中，输入要用于保存Client的存储路径；

  9. 生成完成后，在该目录中会出现名为【FORCEPOINT-ONE-ENDPOINT-Mac.zip】、【FORCEPOINT-ONE-ENDPOINT-x32.exe】和【FORCEPOINT-ONE-ENDPOINT-x64.exe】的Client安装程序。

# 安装Forcepoint Endpoint客户端

一． 安装客户端前的准备工作  因安装Forcepoint Endpoint需重新启动电脑，所以在安装前请保存当前工作，并关闭所有打开的程序。  关闭杀毒软件及其它安全软件。  Mac电脑需要关闭SIP。

关闭杀毒软件，以360安全卫士为例： 在系统右下角的系统托盘上找到360图标 ，右键单击360安全卫士图标，点击“退出”。

出现如下关闭确认框，直接点击“确认”，关闭360

Mac电脑关闭SIP： 关闭Mac SIP系统完整性保护 重启电脑，在重启的过程中按住 Command + R 组合键进入到系统恢复工具界面，然后点击“实用工具”选项卡中的“终端”

进入终端以后，在终端我们输入“csrutil status”然后回车，查询SIP状态，输入“csrutil disable”按下回车，看到下面的提示成功以后我们就解除系统的权限控制了，然后重启电脑，就好了。

二． Windows手工安装/卸载Endpoint客户端 以64位操作系统为例：  安装

  1. 右击桌面【此电脑】图标，选择【属性】查看操作系统类型；

  2. 拷贝客户端安装包到电脑本地目录，右击安装程序【FORCEPOINT-ONE-ENDPOINT-x64.exe】，选择以管理员身份运行；

  3. 在弹出的安装窗口中，点击【Next】按钮；

  4. 在“Subscription Agreement”页面，选择【I accept the terms in the subscription agreement】单选框，并点击【Next】按钮；

  5. 在“Destination Folder”页面中，选择要安装Endpoint Client的安装路径，并点击【Next】按钮；

  6. 在“Ready to install the Program”页面，点击【Install】按钮；

  7. 安装完成后，点击【Finish】按钮，并确认客户端重新启动。

  8. 重启完成后，可以在任务栏右下角看到一个绿色【F】标志的图标，双击可以打开客户端界面。

在其中确认“Forcepoint DLP服务器”的IP地址是否正确，【连接状态】正常情况下应该是“已连接”，而在【端点设置】部分的“状态”应该是【已启用】。

 卸载

  1. 打开添加删除程序，右击【FORCEPOINT ONE ENDPOINT】，选择【卸载】；

  2. 提示选择【是】；

  3. 输入卸载密码；

  4. 卸载完成，根据需要选择是否重启；

三． Mac手工安装/卸载Endpoint客户端  安装

  1. 拷贝客户端安装包到电脑本地目录，右击安装程序【FORCEPOINT-ONE-ENDPOINT-Mac.zip】解压缩安装文件；

  2. 打开解压后的【Endpointinstaller】文件夹，双击【WebsenseEndpoint.pkg】；

  3. 在弹出的安装窗口中，提示选择【继续】；

  4. 在安装向导界面，单击【继续】；

  5. 在“软件许可协议”界面，单击【继续】；

  6. 在“同意许可协议”界面，单击【同意】；

  7. 在“安装磁盘”界面，单击【继续】；

  8. 在“安装类型”界面，单击【安装】；

  9. 输入用户名和密码，点击【安装软件】；

  10. 安装完成后提示注销，单击【LogOff】；

 卸载

  1. 单击Mac图标->系统偏好设置；

  2. 在“系统偏好设置”中单击【Forcepoint图标】；

  3. 如果Forcepoint图标为灰色，请确认用户是否具有卸载权限。或者后台是否有其他安全软件，比如Xserver之类的管理系统。在Endpoint界面中单击【Uninstall Endpoint】图标；

  4. 然后输入用户名和密码，点击【好】。

  5. 卸载完成，右上角F图标消失。

四． 通过AD域组策略来安装Endpoint客户端 推送Client程序

  1. 复制WebsenseEndpoint_32bit.exe和WebsenseEndpoint_64bit.exe到服务器上的共享目录中，并且保证该目录能够通过UNC路径进行访问；
  2. 新建名为Installmsi.bat的批处理文件（bat文件名可以自行选择），输入以下内容；

其中\198.87.116.129为核心服务器主机IP地址，\198.87.116.129\endpoint为UNC共享路径；/v”/quiet /norestart”为安装参数；

  1. 在AD域控服务器上，选择【程序】—【管理工具】—【Active Directory用户和计算机】；

  2. 在要部署客户端的OU上右击选择【属性】；

  3. 在打开的属性窗口中选择【组策略】页面，新建一个名为software的组策略，并点击【编辑】按钮；

  4. 在组策略编辑器中，选择【计算机配置】—【Windows设置】—【脚本(启动/关机)】，双击其中的启动项；

  5. 在打开的“启动属性“对话框中点击【添加】按钮；

  6. 在“添加脚本”对话框中，点击右边的【浏览】按钮；

  7. 将上文中生成的installmsi.bat文件复制到startup目录中，选择其并点击【打开】按钮；

  8. 保存配置；

  9. 点击【确定】按钮完成启动脚本配置；

  10. 在AD域控制器上运行gpupdate /Force命令强制刷新组策略，如果要求重新启动请输入Y并回车。

  11. 重新启动客户终端，则在其重新启动时会自动安装Websense Endpoint；

  12. 安装完成后客户端的状况。

五． 确认是否安装完成

  1. 重启完成后，可以在任务栏右下角看到一个绿色【F】标志的图标，双击可以打开客户端界面。

在其中确认“Forcepoint DLP服务器”的IP地址是否正确，【连接状态】正常情况下应该是“已连接”，而在【端点设置】部分的“状态”应该是【已启用】。

  1. 在TRITON服务器上，可以在【Main】—【Status】—【Endpoint Status】部分中看到当前的Endpoint的状态。

需排除的其他安全软件： 如用户安装了其他安全软件，需配置例外。这里举例360安全卫士、360杀毒、赛门铁克Sep。 一． 360安全卫士 打开360安全卫士主界面

单击上方”木马查杀”选项

单击下方”信任区”按钮

单击右下角”添加目录”按钮 选择Websense EndPoint安装文件夹

确定，排除成功。

二． 360杀毒软件 打开360杀毒软件界面

单击右上角”设置”按钮,出现如下界面： 单击左边栏”文件白名单” 选中右边”添加目录”按钮

选择Websense EndPoint安装文件夹

确定。排除成功。

三． 赛门铁克SEP 右键单击状态栏赛门铁克图标 选择打开Symantic Endpoint Protection

出现赛门铁克配置界面：

选择左边栏更改配置，右边的集中式例外->配置设置按钮 单击添加->安全风险例外->文件夹 选择Websense EndPoint安装文件夹，确认。

再添加->TruScan主动式威胁扫描例外 选择Websense EndPoint安装文件夹内的文件，确认。循环添加多个。
